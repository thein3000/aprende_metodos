{{ method.name }}

Es un método abierto que emplea una fórmula para predecir la raíz.

Está fórmula puede desarrollarse con el método de punto fijo, llamado también sustitución sucesiva, interación de punto fijo o interación simple de punto fijo.

Sea la ecuación general f(×) = 0; de la cual se desea encontrar una raíz real.

1) Consiste en transformar algebraicamente f (x) = 0 a la forma equivalente x g(x).




2) Una vez que se ha determinado una fórmula fórmula equivalente, hay que examinar una sola raíz, esta se puede hacer por observación directa de la ecuación.

Se denota el valor de exploración ó valor de incio como x0.

3) Encontrando x0 se evalúa g(x) en x0 denotándose el resultado de esta evaluación como x; esto es g (x0) = x1.

La utilidad de x = g(x) es proporcionar una fórmula para predecir un nuevo valor de x en función del valor anterior de laisma. De esta manera, dado un valor inicial para la raíz xi.

La ecuación x = g(x) se utiliza para obtener una nueva aproximación xi+1 = g (xi).



Localizar la raíz de f(x) = e-x - x, la función se puede separar directamente y expresarse en forma xi+1 = g(xi). x i+1 = e-xi empezando con un valor inicial x0 = 0. Se aplica esta ecuación iterativa para calcular.
